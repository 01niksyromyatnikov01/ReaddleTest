При решении был использован алгоритм возведения матрицы в степень (алгоритм детально описан в викиучебнике), 
который показывает асимптотику (nlogn)
Также можно было просто забить срез из 30 первых чисел последовательности Фибоначчи, ведь это максимум, 
который может ввести пользователь при одной ошибке на каждом десятке. А при выполнении просто выбирать значения по индексу. 
Сложность такого алгоритма была бы постоянной: О(1).
